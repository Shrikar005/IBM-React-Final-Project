A simple React + Redux houseplant shop built for a front-end assignment.

## Features
- Landing page with background, company paragraph, and Get Started button.
- Product listing with 6 unique plants grouped into 3 categories.
- Add to Cart system using Redux Toolkit.
- Header with live cart count and navigation.
- Cart page with quantity controls, delete, total cost, and checkout.

## Installation
```bash
npm install
npm start
```

## Deployment
```bash
npm run deploy
```
Then visit: https://<your-username>.github.io/plant-shop
